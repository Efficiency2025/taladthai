## ภาพรวม / Overview

ระบบนี้ใช้สำหรับ**เจ้าหน้าที่** ณ จุดลงทะเบียนงานครบรอบ 30 ปี ตลาดไท เพื่อ:

1. **ค้นหา** ข้อมูลผู้ค้าจากชื่อ, เบอร์โทรศัพท์ หรือเลขที่โต๊ะ
2. **ยืนยัน** สิทธิ์การเข้าร่วมงาน และ**แจก Wrist Band** ตามโซน
3. **อัพเดตสถานะ** แบบ Real-time เพื่อป้องกันการแจกซ้ำ

> **URL ระบบ:** https://taladthai.vercel.app

### วิธีค้นหาผู้ค้า

เจ้าหน้าที่สามารถค้นหาผู้ค้าได้ **3 วิธี**:

| วิธี | ตัวอย่าง | หมายเหตุ |
|------|----------|----------|
| **ชื่อ-สกุล** | `สมชาย ใจดี` | ค้นหาแบบบางส่วน (ไม่ต้องพิมพ์ครบ) |
| **เบอร์โทรศัพท์** | `0812345678` หรือ `081-234-5678` | ลบขีดออกได้ |
| **เลขที่โต๊ะ** | `A141-1`, `D111-1`, `VIP_C151-0X` | ตัวพิมพ์ใหญ่-เล็กไม่สำคัญ |

### ขั้นตอน

```
1. พิมพ์ข้อมูลในช่องค้นหา
2. กด Enter หรือกดปุ่ม 🔍 "ตรวจสอบสิทธิ์"
3. รอผลลัพธ์ (ไม่เกิน 1-2 วินาที)
```

## การแจก Wrist Band

### กรณีที่ 1: ผู้ค้าโต๊ะเดียว

1. ตรวจสอบข้อมูลให้ถูกต้อง (ชื่อ, ตลาด, โซน)
2. กดปุ่ม **"แจก Wrist Band"** (สีส้ม)
3. กล่อง confirm ปรากฏ: **"ยืนยันแจก Wrist Band ให้ [ชื่อ] ?"**
4. กด **"OK"** เพื่อยืนยัน
5. ระบบอัพเดต → หน้าเปลี่ยนเป็น ✅ เขียว

```
⚠️ หากกด "OK" แล้ว — ไม่สามารถยกเลิกได้
    กด "Cancel" เพื่อยกเลิก ก่อนกด OK
```

### กรณีที่ 2: ผู้ค้าหลายโต๊ะ (Multi-table)

เมื่อผู้ค้ามีหลายโต๊ะ ระบบจะแสดง **ปุ่มแยกต่างหากสำหรับแต่ละโต๊ะ**:

```
🪑 โต๊ะ
┌──────────────────────┐  ┌──────────────────────┐
│   A141-1             │  │   A141-2             │
│   ⏳ แจก Wrist Band  │  │   ✅ อนุมัติแล้ว     │
│   [กดเพื่ออนุมัติ]   │  │   [ปิดการใช้งาน]    │
└──────────────────────┘  └──────────────────────┘
```

- **ปุ่มสีปกติ** = รอแจก — กดได้
- **ปุ่มสีจาง + disabled** = อนุมัติแล้ว — กดไม่ได้
- เมื่อ**ทุกโต๊ะ**อนุมัติ → ระบบแสดงแถบ ✅ เขียวเต็มหน้า

### กรณีที่ 3: ผู้ค้าได้รับการอนุมัติไปแล้ว (เจ้าหน้าที่คนอื่น)

ระบบแสดง toast สีเหลือง: **"ผู้ค้ารายนี้ได้รับอนุมัติไปแล้ว"**
→ ไม่ต้องแจก Wrist Band ซ้ำ

## คำถามที่พบบ่อย (FAQ)

### Q: ค้นหาชื่อแล้วไม่พบ — ต้องทำอย่างไร?
**A:** ลองค้นหาด้วยวิธีอื่น:
1. ใช้**เบอร์โทรศัพท์** (แนะนำ — แม่นยำที่สุด)
2. ใช้**เลขที่โต๊ะ** เช่น `A141-1`
3. ตรวจสอบการสะกดชื่อ (เช่น ศ/ส, ช/ฉ)

### Q: ผู้ค้า VIP ค้นหายังไง?
**A:** ค้นหาได้หลายรูปแบบ:
- `VIP_D111-1` (underscore)
- `VIP D111-1` (เว้นวรรค)
- `D111-1` (ตรง ๆ)

### Q: กดอนุมัติแล้วแต่ระบบยังโหลด — ต้องรออีกนานไหม?
**A:** ปกติใช้เวลา 1-3 วินาที หากเกิน 10 วินาที ให้รีเฟรชหน้าเว็บและค้นหาใหม่

### Q: เห็น toast "เกิดข้อผิดพลาด กรุณาลองใหม่" — ทำอย่างไร?
**A:**
1. ตรวจสอบสัญญาณอินเทอร์เน็ต
2. กดปุ่ม "แจก Wrist Band" อีกครั้ง
3. หากยังไม่ได้ — รีโหลดหน้าเว็บ แล้วค้นหาใหม่

### Q: ผู้ค้าคนเดิมมาแสดงตัวอีกครั้ง — ระบบจะแจกซ้ำไหม?
**A:** **ไม่** — ระบบแสดงแถบ ✅ เขียว "เข้าร่วมงานเรียบร้อย" แทนปุ่มแจก Wrist Band
